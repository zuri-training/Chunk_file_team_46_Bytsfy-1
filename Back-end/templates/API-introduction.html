What is a CSV File

A comma-separated values (CSV) file is a delimited text file that uses a comma to separate values. Each line of the file is a data record. Each r ecord consists of one or more fields, separated by commas. The use of the comma as a field separator is the source of the name for this file format.

Why Split

Splitting a file is cutting the data in span parts of desired size. Usually, this allows a large file to fit on multiple smaller supports or to fit in multiple email attachments (or upload chunks) if the file is larger than maximum allowed attachment or upload size.
